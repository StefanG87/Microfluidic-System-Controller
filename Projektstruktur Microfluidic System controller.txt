# Project Structure

Current source layout for the Microfluidic System Controller repository.

```text
/Microfluidic-System-Controller/
|-- main_gui.py                         # Main controller GUI entry point
|-- editor_main_embedded.py             # Embedded editor host used from the main GUI
|-- README.md                           # Project overview and local usage
|-- ARCHITECTURE.md                     # Current architecture and automation contract
|-- TESTING.md                          # Manual validation checklist
|-- GUI_PLAN.txt                        # Historical plan, not authoritative
|-- requirements.txt                    # Python dependencies
|
|-- modules/                            # Runtime GUI, hardware, plotting, export, and runner logic
|   |-- __init__.py
|   |-- gui_window.py                   # Main runtime window: PressureFlowGUI
|   |-- measurement_session.py          # Measurement buffers and export snapshots
|   |-- plot_area.py                    # Live Matplotlib plotting widget
|   |-- pressure_controller.py          # Pressure-controller abstraction
|   |-- valve.py                        # Coil-driven valve abstraction
|   |-- flow_sensor.py                  # Analog flow sensor abstraction
|   |-- fluigent_wrapper.py             # Fluigent SDK discovery/readout wrapper
|   |-- sampling_manager.py             # Shared sampling clock and dialog
|   |-- export_dialog.py                # CSV export dialog and writer
|   |-- csv_exporter.py                 # CSV filename and folder helpers
|   |-- program_contract.py             # Shared editor/runner JSON contract
|   |-- program_runner.py               # JSON program execution logic
|   |-- program_worker.py               # Worker-thread wrapper for the runner
|   |-- polynomial_pressure.py          # Advanced pressure profile helpers
|   |-- rotary_valve_controller.py      # High-level rotary valve controller
|   |-- rotary_valve_widget.py          # Rotary valve UI widget and polling
|   |-- rvm_dt.py                       # Low-level AMF RVM DT protocol helper
|   |-- sensor_zero_dialog.py           # Fluigent zero-selection dialog
|   |-- mf_common.py                    # Shared preferences, profiles, and UI helpers
|   `-- Fluigent/                       # Bundled Fluigent SDK package
|
|-- editor/                             # Standalone editor package
|   |-- __init__.py
|   |-- editor_main.py                  # Standalone editor entry point
|   |-- README.txt                      # Editor-specific overview
|   |-- icons/                          # Editor icons
|   `-- modules/editor/
|       |-- __init__.py
|       |-- editor_joblist.py           # Step list, editing, save/load, undo/redo
|       |-- editor_tasklist.py          # Task palette
|       |-- editor_step.py              # Step data model
|       |-- editor_tasks.py             # Standard task dialogs
|       |-- special_tasks.py            # Advanced task dialogs
|       `-- task_globals.py             # Shared editor device catalogs
|
|-- lookup/                             # Hardware profiles and local preference files
|-- icons/                              # Main GUI icons
`-- Measurements/                       # Runtime CSV export output, ignored by Git
```

Generated build outputs, caches, measurements, and local preference files should stay out of version control.
